# World Cup Bets 2026

Static GitHub Pages version of the Prediction Bank app.

## Publish Changes

After editing files locally:

```powershell
git status
git add .
git commit -m "Update app"
git push
```

GitHub Pages will publish from the `main` branch root.

## GitHub Pages Settings

In the repository:

1. Open Settings.
2. Open Pages.
3. Source: Deploy from a branch.
4. Branch: `main`.
5. Folder: `/ (root)`.
6. Save.

The public URL will look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/world-cup-bets-2026/
```
